# 📚 Conceitos de JavaScript - Alura

Bem-vindo! Esta pasta contém uma coleção de arquivos JavaScript desenvolvidos para praticar e consolidar conceitos fundamentais e avançados da linguagem JavaScript.

## 📂 Estrutura da Pasta

### Conceitos Básicos
- **tipos.js** - Tipos de dados primitivos e referência em JavaScript
- **operadoresMatematicos.js** - Operadores aritméticos e suas operações
- **operadoresComparacao.js** - Operadores de comparação e igualdade
- **condicoes.js** - Estruturas condicionais com if/else
- **condicionais.js** - Variações e práticas com condicionais
- **escolha.js** - Switch case e estruturas de decisão

### Arrays e Métodos
- **arrays.js** - Fundamentos de arrays em JavaScript
- **metodosArrays.js** - Métodos nativos: map, filter, reduce, forEach, etc.
- **spreadRestArrays.js** - Operadores spread e rest em arrays

### Funções e Objetos
- **funcoes.js** - Declaração e uso de funções
- **funcoesGlobais.js** - Funções globais disponíveis
- **callbackHOF.js** - Callbacks e Higher-Order Functions
- **hoisting.js** - Comportamento de hoisting em JavaScript
- **objetos.js** - Criação e manipulação de objetos
- **metodosDeObjetos.js** - Métodos de objetos (Object.keys, Object.values, etc.)

### Recursos Avançados
- **destructuring.js** - Desestruturação de arrays e objetos
- **spreadRest.js** - Operadores spread e rest em geral
- **legibilidade.js** - Boas práticas de código limpo e legível
- **combinandoComparadores.js** - Combinação de operadores de comparação

### Estruturas de Repetição
- **lacoFor.js** - Loops com for
- **enquanto.js** - Loops com while
- **desafioLacoRepeticao.js** - Desafios práticos com loops

### Algoritmos e Lógica
- **bruteforce.js** - Algoritmos de força bruta
- **desafio.js** - Desafios gerais de lógica
- **qualy.js** - Exercícios de prática

### Entrada/Saída e Utilitários
- **entradaTerminal.js** - Leitura de entrada do terminal
- **script.js** - Scripts diversos
- **atividades.js** - Atividades práticas

### Projeto - Calculadora

## 🎯 Objetivo

Esta coleção serve como um repositório de estudos para:
- ✅ Aprender conceitos fundamentais de JavaScript
- ✅ Praticar sintaxe e paradigmas
- ✅ Resolver desafios e problemas lógicos
- ✅ Consolidar conhecimento de ES6+
- ✅ Preparação para projetos mais complexos

## 🚀 Como Usar

Cada arquivo pode ser executado individualmente com Node.js:

```bash
node nomeDoArquivo.js
```

Para o projeto calculadora:
```bash
cd calculadora
node index.js
```

## 📝 Notas

- Os arquivos estão organizados por nível de complexidade
- Recomenda-se estudar na sequência: tipos → operadores → condicionais → funções → arrays → objetos → avançados
- Cada arquivo contém exemplos práticos e comentários explicativos
- Sinta-se livre para modificar e experimentar o código!
